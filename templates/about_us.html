About Us

We are a group of three Deep Learning and Convolutional Neural Network enthusiasts wanting to learn about Deep Learning and Convolutional Neural Networks. This passion has inspired us to attempt to solve the complex problem of extracting book details from their covers.

This Project is has allowed us to learn more about Deep Learning and Convolutional Neural Network including models, algorithms and the theory behind them. This project has also allowed us to learn more about building graphical user interfaces, database storage and utilising Google APIs.

For that reason, we believe this project is a success! Allowing us to gain a plethora of knowledge and understanding.
