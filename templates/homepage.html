WELCOME to the Book Keepers Website!


This website allows you to upload an Image of a Book Cover, Our model will extract the Genre, the Confidence of the prediction and filename of the image using machine learning. The Book Predictions will then be stored in a database for you to access. The application will use OCR to extract text from the book cover and run it through Googles Books API to gather the information which will also be stored in a database
